# Core Provider

Enum class for supported model providers.

| Provider     | Description          | Value               |
| ------------ | -------------------- | ------------------- |
| **Gemini**   | Google Gemini models | `Provider.GEMINI`   |
| **OpenAI**   | OpenAI models        | `Provider.OPENAI`   |
| **DeepSeek** | DeepSeek models      | `Provider.DEEPSEEK` |
